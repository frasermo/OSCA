# Safety & Handling

- **Electrical:** Use grounded outlets and supplied power adapters; disconnect power before modifications.
- **Optical:** Avoid direct viewing of LEDs; use eye protection for high-intensity/UV.
- **Mechanical:** Secure the base on a stable table; handle rails and mounts carefully.
- **Transport:** Use supplied Pelican cases; avoid stacking weight on cases.
- **Tools:** Use only supplied tools:
    - Phillips screwdriver (size PH1) #T01
    - Hex key driver 2.0mm #T02
    - Hex key driver 2.5mm #T03
    - Hex key driver 3.0mm #T04
